47 /**
48  * @brief Create a console window.
49  * @details A console window allows text to be written.
50  * @note Text in a console window supports newlines and will wrap text as required.
51  * @return NULL if there is no resultant drawing area, otherwise a window handle.
52  *
53  * @param[in] gc The GConsoleObject structure to initialise. If this is NULL the structure is dynamically allocated.
54  * @param[in] pInit The initialization parameters to use
55  *
56  * @note The drawing color and the background color get set to the current defaults. If you haven't called
57  * @p gwinSetDefaultColor() or @p gwinSetDefaultBgColor() then these are White and Black respectively.
58  * @note The font gets set to the current default font. If you haven't called @p gwinSetDefaultFont() then there
59  * is no default font and text drawing operations will no nothing.
60  * @note On creation even if the window is visible it is not automatically cleared.
61  * You may do that by calling @p gwinClear() (possibly after changing your background color)
62  * @note A console does not save the drawing state. It is not automatically redrawn if the window is moved or
63  * its visibility state is changed.
64  *
65  * @api
66  */

117 /**
118  * @brief Print a formatted string at the cursor position in the window. It will wrap lines as required.
119  * @details This function implements a minimal printf() like functionality
120  * The general parameters format is: %[-][width|*][.precision|*][l|L]p.
121  * The following parameter types (p) are supported:
122  * - <b>x</b> hexadecimal integer.
123  * - <b>X</b> hexadecimal long.
124  * - <b>o</b> octal integer.
125  * - <b>O</b> octal long.
126  * - <b>d</b> decimal signed integer.
127  * - <b>D</b> decimal signed long.
128  * - <b>u</b> decimal unsigned integer.
129  * - <b>U</b> decimal unsigned long.
130  * - <b>c</b> character.
131  * - <b>s</b> string.
132  * @note Uses the current foreground color to draw the string and fills the background using the background drawing color
133  *
134  * @param[in] gh The window handle (must be a console window)
135  * @param[in] fmt The format string (as per printf)
136  * @param[in] ... The format string arguments.
137  *
138  * @api
139  */
