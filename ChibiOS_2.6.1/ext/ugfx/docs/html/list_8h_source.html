70 /**
71  * @brief Create a list widget
72  *
73  * @note The drawing color and the background color get set to the current defaults. If you haven't called
74  * @p gwinSetDefaultColor() or @p gwinSetDefaultBgColor() then these are Black and White.
75  * @note The font gets set to the current default font. If you haven't called @p gwinSetDefaultFont() then
76  * there is no default font and text drawing operations will not display anything.
77  * @note A list remembers its normal drawing state. If there is a window manager then it is automatically
78  * redrawn if the window is moved or its visibility state is changed.
79  * @note The list contains no elements after creation.
80  * @note A slider supports mouse, toggle. Note: toggle only works correctly for single-select lists.
81  * @note When assigning a toggle, only one toggle is supported per role. If you try to assign more than
82  * one toggle to a role, it will forget the previous toggle. Two roles are supported:
83  * Role 0 = toggle for down, role 1 = toggle for up
84  *
85  * @param[in] widget The GListObject structure to initialize. If this is NULL, the structure is dynamically allocated.
86  * @param[in] pInit The initialization parameters to use
87  * @param[in] multiselect If TRUE the list is multi-select instead of single-select.
88  *
89  * @return NULL if there is no resulting drawing area, otherwise a window handle.
90  *
91  * @api
92  */
93 GHandle gwinListCreate(GListObject *widget, GWidgetInit *pInit, bool_t multiselect);

244  * @pre GWIN_NEED_LIST_IMAGES must be set to true in your gfxconf.h
245  *
246  * @param[in] gh The widget handle (must be a list handle)
247  * @param[in] item The item ID
248  * @param[in] pimg The image to be displayed or NULL to turn off the image for this list item.
249  *
250  * @note The image should be up to 4 x (the font height) and a width of (the font height).
251  * The 1st (top) part of the image is displayed for a selected item.
252  * The 2nd part of the image is displayed for an unselected item.
253  * The 3rd part of the image is displayed for a disabled selected item.
254  * The 4th part of the image is displayed for a disabled unselected item.
255  * If the image is less than 4 times the font height then the image use is collapsed
256  * into the available height. For example, an image that equals the font height will use
257  * the same image for all four states.
258  * @note The image is only displayed while it is open. It is up to the application to open
259  * the image.
260  * @note The same image can be used on more than one list item.
261  * @note Images are aligned with the top (not the baseline) of the list item.
262  * @note When any item in the list has an image attached, space is allocated to display
263  * the images even if the image is closed or has been removed by calling @p gwinListItemSetImage()
264  * with a NULL image or by calling @p gwinListItemDelete(). The only way to turn-off the image area
265  * for this list is to call gwinListDeleteAll().
266  *
267  */
